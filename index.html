Welcome to SCUMM - The Game
An homage to the LucasArts adventures of the 1990s.
Use the left and right arrow keys to move.
Type the on-screen commands to interact with the objects you see.
Alternately, click a command followed by an object.
The "use" command requires the "use X on Y" format.
Type "look" by itself to list all on-screen items.
Click anywhere to begin!
